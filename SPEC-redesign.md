# global-news-24h — Readhub 风格重建规范

## 目标

将 global-news-24h 从现有设计改版为 Readhub 风格的信息流聚合站，极简、高效、科技感。

---

## 设计语言

### 色调
- 背景：`#ffffff`
- 文字主色：`#1a1a1a`
- 次级文字：`#8a8a8a`
- 品牌蓝：`#3467a8`
- 分割线：`#f0f0f0`
- 悬浮操作：`rgba(0,0,0,0.06)`

### 字体
- 主字体：系统默认（-apple-system / BlinkMacSystemFont / "Segoe UI"）
- 标题：16px / bold
- 时间/来源：12px / 灰色
- 标签：12px / 蓝色品牌色

### 间距
- 行高：1.8
- 内容区 padding：0 16px
- 每条间距：0（用分割线隔开）

---

## 页面结构

### Header（固定顶部）
- 左侧：Logo "Readhub"（加粗）
- 中间：空白（留对称）
- 右侧：搜索图标 🔍
- 高度：48px，白色背景，底部一根 1px 分割线

### 导航栏（固定顶部，header 下方）
- 横向滚动，带 ps-y 隐藏滚动条
- 项目：热门话题 | 每日早报 | 排行榜 | AI | 更多
- 选中项：品牌蓝 + 2px 下划线
- 高度：44px

### 主内容（无限滚动信息流）
- 无分割线列表，每条之间 1px 分割线
- 右侧悬浮：浮动操作栏（分享/收藏/反馈）

### 底部（无 TabBar，纯阅读体验）

---

## 每条新闻格式

```
[类型标签] [标题 — 加粗，1-2行]        [时间：N分钟前]
[来源：Reuters · 1分钟前]
```

- 标题：`16px bold #1a1a1a`
- 时间：`12px #8a8a8a`，右侧对齐
- 类型标签：`12px #3467a8`，标题左侧（如果有）
- 来源：`12px #8a8a8a`，标题下方

---

## 浮动操作栏（右侧）
- 半透明圆形按钮，毛玻璃效果
- 功能：分享链接、回到顶部、反馈
- 固定在视口右侧中央

---

## 分类页（导航点击后）

### 热门话题
- 默认标签，All / 科技 / 商业 / AI
- 子标签横向滚动

### 每日早报
- 每天早上 8:00 自动推送
- 展示今日已读的摘要

### AI
- 下拉展开子分类：LLM / 机器人 / 芯片 / 工具

### 排行榜
- 阅读量 / 评论数排序

---

## 技术实现

- 纯静态 HTML + CSS + Vanilla JS
- GitHub Actions 每 30 分钟自动抓取
- 数据源：`src/sources.json`（已有 RSS 源）
- 翻译：MyMemory 免费 API → 改为阿里云百炼（或保留）
- 部署：GitHub Pages

---

## 文件结构

```
index.html          — 主页面
css/
  style.css        — 所有样式
js/
  app.js          — 前端逻辑
.github/
  workflows/
    update-news.yml — 自动更新
src/
  fetch_news.py   — 抓取脚本（保持不变）
  sources.json    — RSS 源配置
```

---

## 工作分解

1. **SPEC 本文档** ← 当前
2. **index.html + style.css** — 重构 UI（独立任务）
3. **js/app.js** — 导航切换、滚动加载（独立任务）
4. **GitHub Actions** — 自动部署（已在 `update-news.yml`）
5. **测试 + 上线**
